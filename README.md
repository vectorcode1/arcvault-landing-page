## ✨ Features

- Fully responsive — mobile, tablet, desktop
- Animated hero section with staggered text reveals
- Scroll-triggered fade-in animations
- Interactive project gallery with hover overlays
- Services grid with hover effects
- Contact form UI
- Mobile hamburger menu with smooth transitions
- Grain texture overlay for depth
- Custom gold scrollbar

## 🛠️ Tech Stack

- **HTML5** — semantic structure
- **Tailwind CSS** (CDN) — utility-first styling
- **Vanilla JavaScript** — scroll animations, mobile menu
- **Google Fonts** — Cormorant Garamond + Montserrat
- **Unsplash** — placeholder images

## 🚀 Getting Started

No build step needed. Just open the file:

```bash
git clone https://github.com/vectorcode1/arcvault-landing-page
cd arcvault-landing-page
open index.html
```

Or drag `index.html` into your browser.
